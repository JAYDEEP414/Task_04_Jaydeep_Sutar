# Project 4: Data Visualization using Power BI

## 📌 Objective
The objective of this project was to create an interactive and visually appealing Power BI dashboard to analyze e-commerce sales data and generate meaningful business insights.

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- Data Visualization
- Dashboard Design

---

## ✅ Tasks Performed
- Imported cleaned e-commerce dataset into Power BI
- Created KPI cards for:
  - Total Revenue
  - Total Orders
  - Average Order Value
  - Total Quantity Sold
- Built interactive visualizations:
  - Product Revenue Analysis
  - Monthly Revenue Trend
  - Payment Method Distribution
  - Order Status Distribution
  - Revenue by Country
  - Applied dashboard formatting and layout design
- Generated business insights through visual storytelling

---

## 📊 Dashboard Features
- Interactive Filters
- KPI Metrics
- Revenue Trend Analysis
- Product Performance Insights
- Payment Method Analysis
- Order Status Tracking
- Country-wise Revenue Analysis

---

## 📂 Files Included
- `Project 4.pbix`

---

## 📈 Key Insights
- Chair generated the highest revenue among all products.
- Online was the most commonly used payment method.
- Revenue showed strong monthly fluctuations and growth trends.
- Cancelled orders were higher compared to delivered orders.
- Electronics and furniture products contributed significantly to total revenue.
